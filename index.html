Curso css

El formato es:

selector {
propiedad: valor;
}

CSS Selector

  • Universal:* Selecciona a todos los elementos.
    • De tipo:por ejemplo: h6{color: #28e;}
  • Clases:con el atributo clase en el dom. Se usa un punto para seleccionar.
  • ID: con el atribudo id en el dom. Se usa el signo de gato (#) para seleccionar.
  • Por Atributo:Puede ser como name="nombre", se pone: [name="nombre"]
  • Descendiente:Se seleccionan etiquetas anidadas, por ejemplo: li b {color: green;} También funciona con clases.
  • Pseudo-clases:Se usa en eventos de clases como active o hover.
